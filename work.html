    Artificial Intelligence using Deep Learning with Neural Networks

    This project was carried out for a web APP for brain tumor detection by using Neural Networks and Deep Learning deployment for images recognition. This work was intended to help doctors from several disciplines as neurologists, oncologists, radiologist, neurosurgeons and magnetic resonance image analysts.
    For this work, public databases were used to train the neural network designed to detect the 3 most common types of brain tumor in Spain.

    PLC Process Automation

    This work was executed in order to automate a bottling beer process line. To do this, it has been used UNITY PRO, a Schneider Electric software for their PLC products. Main target was to develop a process control by using automated automatons, timers, electric valves, relays, among others.
    Human machine interface was also implemented during the making of this project, the final deliver let the user have absolute control over the process and active elements of the system.
